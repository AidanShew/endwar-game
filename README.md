# End War (Working Title)
***Check out the GitHub Wiki for more information about the content release roadmap, this project's expectations, some notes from me, and more.***

### Controls: *Advanced Combat*
- Use WASD to move your ship
- Use 'J' to fire slightly left, 'I'/Spacebar to fire straight up, 'L' to fire slightly right
- Return to the main menu with the Escape key. **Does not pause gameplay**

### Horde Mode: *Ranked*
Prove yourself in the "Arena", fight waves of enemies, every 5 rounds will activate a boss battle against a Whirlpool. Good luck.

Your player stats will be tracked via the main menu, earn your rank:
- Noob (Die within the first round)
- Cadet (Survive the first round)
- Soldier (Survive at least 5 rounds, including defeating the first Whirlpool)
- Hero (Survive more than 10 rounds)
- Champion (Survive more than 20 rounds... Godspeed)
