# Update 1.0.1 *Hot fix*
### Main Changes
- Fixed the issue where player speed would decrease dramatically after respawn.
- (For Developers) Fixed an issue where the GameMaker project file wouldn't open.
- In Horde Mode, enemies will no longer spawn from the bottom of the screen.
- Fixed issue where enemy Soldier would spawn repeatedly in the same area.
- Enemy Soldiers will now spawn one-by-one, waiting until the previous has entered the arena.
- Fixed issue if player dies while fighting Whirlpool, the next round would commence during the boss battle.
- Fixed issue if player collides with Follower, Whirlpool will not replenish Follower amount.
- Fixed issue where incorrect rank would appear on main menu.
- When player dies during boss battle, the Whirlpool will now respawn appropriately instead of remaining in the middle of the screen.
- Added player gun fire sound effects as well as explosion sound effects.

### Miscellanous Changes
- Updated README for GitHub project.
- Cleaned up some of the code for the project, made it more readable and understandable.

### Known Issues
- Sometimes Whirlpool rotation will reset, making it difficult to strategize. This is rare but is an issue I am aware of.
